### 喜欢的就点个星星吧
**gradle导入：**

* 1.项目 的build.gralde中添加

```

	allprojects{

		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

```
* 2.添加依赖
```
dependencies {
	     implementation 'com.github.DingMouRen:LayoutManagerGroup:1e6f4f96eb'
	}
```
